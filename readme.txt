== Description ==
Acest plugin va permite sa alegeti posturile tv favorite si sa le afisati sub forma unui widget, pe blogurile sau site-urile personale.

== Installation ==
Pasii de urmat pentru instalarea pluginului:

    1. Descarcati pluginul, dezarhivati si urcati directorul tv-online in folderul /wp-content/plugins/.
    2. Intrati in partea de administrare a blogului, Plugins si activati pluginul "TvWidget"
    3. In meniul principal veti gasi butonul "Tv Widget" de unde puteti administra categoriile si radiourile care doriti sa fie afisate pe blogul vostru
    4. Daca doriti sa afisati posturile din anumite categorii bifati categoriile respective si apoi apasati butonul Listeaza Posturi Tv. Din lista afisata bifati posturile care doriti sa le afisati in sidebarul blogului si apoi bifati "Salveaza Posturi Tv".
    5. Acum navigati la Appearance -> Widgets si alegeti pozitia widgetului in sidebar.
